# 📊 DLF Real Estate Analysis Dashboard (Power BI)

## 🚀 Project Overview

This project presents a comprehensive analysis of DLF Limited’s real estate business using Power BI. The dashboard focuses on understanding how the company operates across different segments such as development pipeline, inventory position, sales performance, and geographic contribution.

The goal of this project is to convert raw data into meaningful insights that reflect both current performance and future growth potential.

---

## 📌 Key Insights

* A significant portion of revenue is driven by the Gurugram region, highlighting strong regional concentration.
* The company maintains a large development potential (~220 msf), indicating long-term expansion capability.
* Around 41 msf of projects are currently under construction, forming a strong near-term pipeline.
* Unsold inventory declined over time but saw a sharp rise in FY25 due to new luxury project launches.
* Project deliveries are gradually catching up with new launches, reflecting improved execution.

---

## 📊 Dashboard Highlights

The dashboard is divided into multiple analytical sections:

* **Revenue Analysis**

  * Geographic revenue split
  * Future sales potential by region

* **Project Pipeline**

  * Total development potential
  * Under construction projects
  * Delivered projects since IPO

* **Market Trends**

  * Unsold inventory trend
  * Rental yield growth
  * Housing segment shift (affordable vs luxury)

* **Operational Performance**

  * New launches vs completions
  * RERA project status distribution
